# 语言基础

现在你已经安装了 Toka 并编写了第一个程序，让我们深入了解这门语言的核心概念。

## Hello World 回顾

最简单的 Toka 程序只需要两样东西：一个标准库的 `import` 和一个 `main` 函数：

```toka
import std/io::println

fn main() -> i32 {
    println("Hello, Toka!")
    return 0
}
```

运行方式：

```bash
toka run hello.tk
```

## 核心原则

Toka 围绕几个核心原则设计：

1. **默认安全** — 变量默认不可变，除非显式标记。
2. **无隐藏运行时** — 你写什么就执行什么。没有垃圾回收器，没有虚拟机。
3. **类 C 性能** — 基于 LLVM 20，Toka 生成优化的原生代码。
4. **现代语法** — 简洁、可读性强，配备强大的模式匹配和错误处理。

## 注释语法

Toka 中的注释使用标准的 `//` 语法：

```toka
// 这是单行注释

/* 这是
   多行注释 */
```
